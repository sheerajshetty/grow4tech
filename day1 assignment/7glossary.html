Glossary of Web Development Terms

CSS
Cascading Style Sheets - A language used to style the appearance of web pages.
The CSS code defines the fonts, colors, and layout of this page.
JavaScript
A scripting language used to add interactivity to web pages.
JavaScript can be used to create animations, forms, and other dynamic elements.
URL
Uniform Resource Locator - The address of a web page.
The URL of this page is https://example.com/glossary.html.
